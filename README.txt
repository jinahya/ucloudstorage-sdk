< ucloudstorage java sdk >
ucloudstorage java sdk는 ucloudstorage 서비스에 대한 java api를 제공한다.
본 sdk를 이용하여, ucloudstorage의 다양한 응용프로그램을 개발할 수 있다.

< ucloudstorage java sdk 구성 >
ㅇ lib - ucloudstorage java sdk를 포함한 다양한 third-party library
ㅇ src - ucloudstorage java sdk 소스 파일
ㅇ javadoc - ucloudstorage sdk java document
ㅇ samples - 샘플 소스파일

< 개발 환경 >
ㅇ Windows7 64bit
ㅇ JDK 1.6
